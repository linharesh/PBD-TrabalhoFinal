Trabalho final da disciplina
-> Tema livre (sob aprovação)
-> Modelo conceitual
-> Esquema no SGBD escolhido
-> Cada esquema deve conter no mínimo 25 tableas
-> Cada grupo deve propor no mínimo 5 stored procedures ou funções
-> Cada grupo deve criar índices para o BD criado e justificar a criação de cada um dos índices
-> Cada grupo deve implementar no mínimo 3 triggers (gatilhos)
-> Cada grupo deve implementar no mínimo 2 views

-> Uso de view materializada
-> Fragmentação de tabelas
